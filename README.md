## Market Price Bot for Home Assistant OS

### Возможности
- Telegram-бот
- Ozon + Wildberries
- Просто кидаешь ссылку
- Подтверждение
- Интервалы 3/6/9/12 часов
- Уведомления
- Графики

### Установка
1. Settings → Add-ons → Repositories
2. Вставь GitHub URL
3. Установи Add-on
4. Введи Telegram Token и Chat ID
5. Start
